# PChome-search
https://shopping.pchome.com.tw/

data裝載資料
1. keyword used: 使用者所打的關鍵字 (ex:NBA球衣)
2. # of SKU: 關鍵字的產品總數
3. Page: 產品位於第幾頁
4. Description: 產品說明
5. PrdCode: 產品ID
6. position_section: 在搜尋中，該商品位於第幾筆 (由上往下數)
7. Selling Price: 產品販售價格
8. List Price: 產品原價
9. Brand: 產品品牌
10. route: 產品路徑
11. rtime: 資料爬取時間
12. Out of Stock: 是否賣完 (1:賣完；0:沒賣完)
